How to install
===========================


Requirements
------------

-  Python 3.6+
-  pip/pipenv (used to get the rest of the requirements)

.. _how_to_install:

Install
-------

.. code:: bash

   git clone https://github.com/sebastianpfischer/investment-portfolio.git
   cd investment-portfolio
   pip install .

With `Pipenv <https://github.com/pypa/pipenv>`__ :

.. code:: bash

   git clone https://github.com/sebastianpfischer/investment-portfolio.git
   cd investment-portfolio
   pipenv install --dev
   pipenv shell
